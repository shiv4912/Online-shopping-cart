🛒 Online Shopping Cart

A full-stack Online Shopping Cart application built using Spring Boot (backend) and HTML, CSS, JavaScript (frontend). It allows users to view products, add items to the cart, and calculate the total dynamically.

🚀 Features
View products from backend API
Add items to cart
Dynamic total price calculation
Simple and responsive UI
🛠️ Tech Stack
Backend: Spring Boot, Java, JPA, MySQL
Frontend: HTML, CSS, JavaScript
⚙️ Run Locally
Start backend (Spring Boot)
Open index.html in browser
Ensure API runs at: http://localhost:8080/products
